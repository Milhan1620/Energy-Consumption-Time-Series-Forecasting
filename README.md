# ⚡ Energy Consumption Time Series Forecasting

## 📖 Overview

This project forecasts household electricity consumption using historical power usage data. Multiple forecasting algorithms are implemented and compared to identify the most accurate model for predicting future energy demand.

---

## 🎯 Objectives

- Process time-series energy data.
- Engineer temporal features.
- Compare forecasting models.
- Visualize actual and predicted energy consumption.

---

## 📂 Dataset

**Dataset:** Household Power Consumption Dataset

The dataset contains household electricity measurements collected over time.
